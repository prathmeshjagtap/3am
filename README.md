# 3am Video Library for Awesome Music

---

## Quick Start

Preview Link : https://3am-music.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/prathmeshjagtap/3am.git
```

Go to the project directory

```bash
  cd 3am
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features Available.

- Home Page with all videos and Filters by category and search
- Video Listing Page
- Like Feature
- History
- Watch Later Functionality
- Playlist
  - Add to Playlist
  - Create Playlist
  - Delete Playlist
- Authentication
  - Login
  - Sigup
  - Guest Login
- Fully Resposive
- Sort By category
- Search Videos

---

## Tech Stack

**Client:** React, React Context API + useReducer , React Router v6 , Easy UI for CSS

**Server:** Mockbee.
